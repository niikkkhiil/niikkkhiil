## 🏆 Featured Project

<div align="center">

### 🛡️ Self-Healing DevOps Agent

*Autonomous Docker container monitoring → AI-powered diagnosis → smart remediation*

</div>

**The pitch:** A multi-agent AI system that watches Docker infrastructure 24/7, reads logs using LLMs, identifies failure types (OOM, port conflicts, disk full, crashes), and applies targeted fixes — eliminating manual remediation entirely.

**Why it's interesting:**
- 🧠 **3 specialist agents** — Monitor, Analyzer, Executor (built on CrewAI)
- 💾 **FAISS vector memory** — past incidents resolve instantly, no LLM call needed
- 📊 **Langfuse observability** — every agent decision is traced and inspectable
- 🚀 **~70% token cost reduction** via memory caching on recurring incidents
- 🔧 **Smart error detection** — 7 distinct error types, each with a tailored fix
- 📦 **Dockerised + CI/CD** — GitHub Actions auto-builds and pushes to Docker Hub

**Stack:** `Python` `CrewAI` `Groq (Llama 3.3 70B)` `FAISS` `Langfuse` `MCP` `Docker` `GitHub Actions`
